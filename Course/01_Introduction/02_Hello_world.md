## Перша програма
У цьому розділі обговорюється створення вашої першої програми, запуск інтерпретатора та деякі основні налагодження.

### Запуск Python
Програми на Python завжди запускаються всередині інтерпретатора. 

Інтерпретатор — це «консольна» програма, яка зазвичай запускається з терміналу.
```bash
python3
Python 3.8.10 (default, Sep 28 2021, 16:10:42)
[GCC 9.3.0] on linux
Type "help", "copyright", "credits" or "license" for more information.
>>>
```
У досвідчених програмістів зазвичай не виникає проблем із використанням інтерпретатора
таким чином, але він не настільки зручний для початківців. Ви можете використовувати
середовище, яке надає інший інтерфейс для Python. Це ок,
але вміти запускати і користуватися терміналом Python - це все ще корисна навичка.

### Інтерактивний режим
Коли ви запускаєте Python, ви відкриваєте інтерактивний режим, в якому можна експериментувати.

Якщо ви почнете вводити команди, вони запускатимуться негайно. Циклу редагування/компіляція/запуск/дебаг не існує.

```python
>>> print('hello world')
hello world
>>> 37*42
1554
>>> for i in range(5):
...     print(i)
...
0
1
2
3
4
>>>
```

Цей так званий цикл read-eval-print-loop (або REPL) дуже корисний для дебагінгу та експериментації.

**АСТАНАВІТЄСЬ**: якщо ви не можете зрозуміти, як взаємодіяти з Python, припиніть те, що ви робите, і з’ясуйте, що і до чого. Якщо ви використовуєте IDE, інтерпретатор може бути прихованим за меню або якимось іншим вікном. У багатьох розділах цього курсу припускається, що ви можете взаємодіяти з Python у терміналі.

Давайте детальніше розглянемо елементи REPL:
- `>>>` значок інтерпретатора для початку команди
- `...` значок інтерпретатора для продовження команди. Введіть порожній рядок, щоб завершити введення, і запустіть.

Символ `...` може відображатися або не відображатися в залежності від вашого середовища. Для цього курсу він показаний у вигляді пробілів, щоб було легше копіювати/вставляти зразки коду.

Символ підкреслення `_` містить останній результат.

```python
>>> 37 * 42
1554
>>> _ * 2
3108
>>> _ + 50
3158
>>>
```

*Так воно працює лише в інтерактивному режимі.* `_` у програмі працює зовсім по-іншому. І взагалі, краще утриматися від його використання в програмі.

### Створення програм
Програми пишуться у файли з розширенням `.py`
```python
# hello.py
print('hello world')
```

Ви можете створювати ці файли за допомогою будь-якого текстового редактора.

### Запуск програм
Запуск програми відбувається в терміналі за допомогою команди `python`. Наприклад, у командному рядку Unix це виглядатиме так:
```bash
bash % python hello.py
hello world
bash %
```

Або з Windows:
```
C:\SomeFolder>hello.py
hello world

C:\SomeFolder>c:\python36\python hello.py
hello world
```

Примітка. У Windows вам може знадобитися вказати повний шлях до інтерпретатора Python, наприклад `c:\python36\python`. Однак, якщо Python встановлено звичайним способом, ви можете просто ввести назву програми, наприклад `hello.py`.

### Приклад програми
Розв’яжемо наступну задачу:
> Одного ранку ви виходите і кладете доларову купюру на тротуарі біля вежі Сірс у Чикаго.
> Кожного дня після цього ви кладете поверх подвоєну кількість банкнот.
> Скільки часу потрібно, щоб стос купюр перевищив висоту вежі?

А ось і рішення:
```python
# sears.py
bill_thickness = 0.11 * 0.001 # Meters (0.11 mm)
sears_height = 442 # Height (meters)
num_bills = 1
day = 1

while num_bills * bill_thickness < sears_height:
    print(day, num_bills, num_bills * bill_thickness)
    day = day + 1
    num_bills = num_bills * 2

print('Number of days', day)
print('Number of bills', num_bills)
print('Final height', num_bills * bill_thickness)
```
Після запуску ви отримаєте наступне:
```bash
bash % python3 sears.py
1 1 0.00011
2 2 0.00022
3 4 0.00044
4 8 0.00088
5 16 0.00176
6 32 0.00352
...
21 1048576 115.34336
22 2097152 230.68672
Number of days 23 
Number of bills 4194304 
Final height 461.37344
```
Взявши цю програму за основу, ви можете дізнатися ряд важливих основних понять про Python.

### Команди
Програма на python - це послідовність команд:
```python
a = 3 + 4
b = a * 2
print(b)
```

Кожна команда закінчується новим рядком. Команди виконуються одна за одною, поки елемент керування не досягне кінця файлу.

### Коментарі
Коментарі - це текст, який буде проігноровано.
```python
a = 3 + 4
# This is a comment
b = a * 2
print(b)
```

Коментарі позначаються `#` і поширюються до кінця рядка.

### Змінні
Змінна - це ім'я для значення. Ви можете використовувати літери (малі та великі) від a до z. А також символ підкреслення `_`. Числа також можуть бути частиною імені змінної, за винятком першого символу.

```python
height = 442 # ок
_height = 442 # ок
height2 = 442 # ок
2height = 442 # не ок
```

### Типи
Змінні не потрібно оголошувати разом із типом значення. Тип пов’язаний із значенням у правій частині, а не з назвою змінної.
```python
height = 442           # An integer (Ціле число)
height = 442.0         # Floating point (Десятковий дріб)
height = 'Really tall' # A string (Текст)
```

Python - це мова з динамічними типами. Сприйнятий «тип» змінної може змінюватися під час виконання програми залежно від поточного значення, призначеного змінній.

### Чутливість до регістру
Python чутливий до регістру. Великі та малі літери вважаються різними літерами. Це всі різні змінні:
```python
name = 'Jake'
Name = 'Elwood'
NAME = 'Guido'
```

Зарезервовані команди мови завжди в нижньому регістрі.
```python
while x < 0:   # OK
WHILE x < 0:   # ПОМИЛКА
```

### Цикли
Оператор `while` запускає цикл.
```python
while num_bills * bill_thickness < sears_height:
    print(day, num_bills, num_bills * bill_thickness)
    day = day + 1
    num_bills = num_bills * 2

print('Number of days', day)
```

Операції під `while` з відступом виконуватимуться до тих пір, поки вираз після `while` є істинним (`true`).

### Відступи
Відступ використовується для позначення груп команд, які поєднуються в один блок. Розглянемо попередній приклад:
```python
while num_bills * bill_thickness < sears_height:
    print(day, num_bills, num_bills * bill_thickness)
    day = day + 1
    num_bills = num_bills * 2

print('Number of days', day)
```
Відступ об’єднує команди внизу як блок повторюваних операцій:
```python
    print(day, num_bills, num_bills * bill_thickness)
    day = day + 1
    num_bills = num_bills * 2
```
Оскільки функція `print()` в кінці не має відступу, вона не належить до циклу. Порожній рядок призначений лише для читабельності. На перебіг програми він не впливає.

### Методика роботи з відступами
* Використовуйте пробіли замість табуляції.
* Використовуйте 4 пробіли на рівень.
* Використовуйте редактор з підтримкою Python.

Єдина вимога Python полягає в тому, щоб відступи в одному блоці були послідовними. Наприклад, це помилка:
```python
while num_bills * bill_thickness < sears_height:
    print(day, num_bills, num_bills * bill_thickness)
        day = day + 1 # ПОМИЛКА
    num_bills = num_bills * 2
```

### Умови
Оператор `if` використовується для постановки умови (якщо щось, то...):
```python
if a > b:
    print('Машина каже ні')
else:
    print('Машина каже так')
```
Можна одночасно перевіряти декілька умов, для цього використовується оператор `elif`:
```python
if a > b:
    print('Машина каже ні')
elif a == b:
    print('Машина каже так')
else:
    print('Машина каже можливо')
```

### Вивід тексту
Функція `print` виводить одну лінію тексту із значень, що були надані функції:
```python
print('Привіт світ!') # Виведе текст 'Привіт світ!'
```
Ви можете використовувати змінні. Виведений текст буде значенням змінної, а не її назвою.
```python
x = 100
print(x) # Виводиться текст '100'
```
Якщо ви надасте більш, ніж одне значення до функції `print` вони виведуться розділені пробілом.
```python
name = 'Jake'
print('My name is', name) # Виведеться текст 'My name is Jake'
```
`print()` завжди додає нову лінію в кінці:
```python
print('Hello')
print('My name is', 'Jake')
```
Виведе наступне:
```code
Hello
My name is Jake
```
Цю нову лінію в кінці можна відключити:
```python
print('Hello', end=' ')
print('My name is', 'Jake')
```
І тепер ми отримаємо:
```code
Hello My name is Jake
```

### Введення даних користувачем
Щоб прочитати рядок введений користувачем, використайте функцію `input()`:
```python
name = input('Enter your name:')
print('Your name is', name)
```
`input()` друкує підказку для користувача та повертає його відповідь. Зручно для невеликих програм, навчальних вправ або простого дебагу. Особливо не використовується для реальних програм.

### Ключове слово pass
Іноді потрібно вказати порожній блок коду. Для нього використовується ключове слово `pass`.
```python
if a > b:
    pass
else:
    print('Computer says false')
```
Це також називається "ноу-оп" оператором. Він нічого не робить. Він служить заповнювачем для команд, можливо, щоб додати їх пізніше.

### Вправа 1.6: Дебаг
Наступний фрагмент коду містить код із проблеми Tower Sears. У ньому також є помилка.
```python
# sears.py

bill_thickness = 0.11 * 0.001    # Meters (0.11 mm)
sears_height   = 442             # Height (meters)
num_bills      = 1
day            = 1

while num_bills * bill_thickness < sears_height:
    print(day, num_bills, num_bills * bill_thickness)
    day = days + 1
    num_bills = num_bills * 2

print('Number of days', day)
print('Number of bills', num_bills)
print('Final height', num_bills * bill_thickness)
```

Скопіюйте та вставте код, який відображається вище, в нову програму під назвою `sears.py`. Під час запуску коду ви отримаєте повідомлення про помилку, яке змушує програму зупинитись:
```code
Traceback (most recent call last):
  File "sears.py", line 10, in <module>
    day = days + 1
NameError: name 'days' is not defined
```

Розуміння повідомлень про помилки є важливою частиною програмування на Python. Якщо ваша програма ламається, останній рядок повідомлення в traceback є фактичною причиною, чому програма зазнала краху. Над цим повідомленням ви повинні побачити фрагмент вихідного коду, а потім ідентифікаційне ім'я файлу та номер рядка.
