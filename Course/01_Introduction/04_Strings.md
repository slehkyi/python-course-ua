# 1.4 –¢–µ–∫—Å—Ç (—Ä—è–¥–∫–∏)
–£ —Ü—å–æ–º—É —Ä–æ–∑–¥—ñ–ª—ñ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω—ñ —Å–ø–æ—Å–æ–±–∏ —Ä–æ–±–æ—Ç–∏ –∑ —Ç–µ–∫—Å—Ç–æ–º.

### –ü—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–Ω—è —Ç–µ–∫—Å—Ç—É
–†—è–¥–∫–æ–≤—ñ –ª—ñ—Ç–µ—Ä–∞–ª–∏ –∑–∞–ø–∏—Å—É—é—Ç—å—Å—è –≤ –ø—Ä–æ–≥—Ä–∞–º–∞—Ö —É –ª–∞–ø–∫–∞—Ö.
```python
# Single quote
a = 'Yeah but no but yeah but...'

# Double quote
b = "computer says no"

# Triple quotes
c = '''
Look into my eyes, look into my eyes, the eyes, the eyes, the eyes,
not around the eyes,
don't look around the eyes,
look into my eyes, you're under.
'''
```

–ó–∞–∑–≤–∏—á–∞–π —Ä—è–¥–∫–∏ –æ—Ö–æ–ø–ª—é—é—Ç—å –ª–∏—à–µ –æ–¥–∏–Ω —Ä—è–¥–æ–∫. –ü–æ—Ç—Ä—ñ–π–Ω—ñ –ª–∞–ø–∫–∏ —Ñ—ñ–∫—Å—É—é—Ç—å –≤–µ—Å—å —Ç–µ–∫—Å—Ç, —É–∫–ª–∞–¥–µ–Ω–∏–π —É –∫—ñ–ª—å–∫–∞ —Ä—è–¥–∫—ñ–≤, –≤–∫–ª—é—á–∞—é—á–∏ –≤—Å–µ —Ñ–æ—Ä–º–∞—Ç—É–≤–∞–Ω–Ω—è.

–ù–µ–º–∞—î —Ä—ñ–∑–Ω–∏—Ü—ñ –º—ñ–∂ –≤–∏–∫–æ—Ä–∏—Å—Ç–∞–Ω–Ω—è–º –æ–¥–∏–Ω–∞—Ä–Ω–∏—Ö (') —ñ –ø–æ–¥–≤—ñ–π–Ω–∏—Ö (") –ª–∞–ø–æ–∫. *–û–¥–Ω–∞–∫ –¥–ª—è –∑–∞–≤–µ—Ä—à–µ–Ω–Ω—è —Ä—è–¥–∫–∞ –Ω–µ–æ–±—Ö—ñ–¥–Ω–æ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ —Ç–æ–π —Å–∞–º–∏–π —Ç–∏–ø –ª–∞–ø–æ–∫, —è–∫–∏–π –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—î—Ç—å—Å—è –¥–ª—è –ø–æ—á–∞—Ç–∫—É —Ä—è–¥–∫–∞.*

### Escape-–∫–æ–¥–∏ —Ä—è–¥–∫—ñ–≤
Escape-–∫–æ–¥–∏ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—é—Ç—å—Å—è –¥–ª—è –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–Ω—è –∫–æ–Ω—Ç—Ä–æ–ª—å–Ω–∏—Ö —Å–∏–º–≤–æ–ª—ñ–≤ —ñ —Å–∏–º–≤–æ–ª—ñ–≤, —è–∫—ñ –Ω–µ –º–æ–∂–Ω–∞ –ª–µ–≥–∫–æ –≤–≤–µ—Å—Ç–∏ –ø—Ä—è–º–æ –Ω–∞ –∫–ª–∞–≤—ñ–∞—Ç—É—Ä—ñ. –ù–∏–∂—á–µ –Ω–∞–≤–µ–¥–µ–Ω–æ –∫—ñ–ª—å–∫–∞ –ø–æ—à–∏—Ä–µ–Ω–∏—Ö escape-–∫–æ–¥—ñ–≤:
```
'\n'      –ù–æ–≤–∞ –ª—ñ–Ω—ñ—è
'\r'      –ï–Ω—Ç–µ—Ä - —Ç–∞–∫, –∫–æ–¥–æ–≤–∞ —Ä–µ–ø—Ä–µ–∑–µ–Ω—Ç–∞—Ü—ñ—è –µ–Ω—Ç–µ—Ä—É
'\t'      –¢–∞–± - —Ç–µ –∂ —Å–∞–º–µ, —â–æ —ñ –∑ –µ–Ω—Ç–µ—Ä–æ–º
'\''      –û–¥–∏–Ω–∞—Ä–Ω—ñ –ª–∞–ø–∫–∏
'\"'      –ü–æ–¥–≤—ñ–π–Ω—ñ –ª–∞–ø–∫–∏
'\\'      –ó–≤–æ—Ä–æ—Ç–Ω–∞ –∫–æ—Å–∞ —Ä–∏—Å–∫–∞
```

### –ü—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–Ω—è —Ä—è–¥–∫—ñ–≤
–ö–æ–∂–µ–Ω —Å–∏–º–≤–æ–ª —É —Ä—è–¥–∫—É –∑–±–µ—Ä—ñ–≥–∞—î—Ç—å—Å—è –≤—Å–µ—Ä–µ–¥–∏–Ω—ñ —è–∫ —Ç–∞–∫ –∑–≤–∞–Ω–∞ "–∫–æ–¥–æ–≤–∞ —Ç–æ—á–∫–∞" Unicode, —è–∫–∞ —î —Ü—ñ–ª–∏–º —á–∏—Å–ª–æ–º. –í–∏ –º–æ–∂–µ—Ç–µ –≤–∫–∞–∑–∞—Ç–∏ —Ç–æ—á–Ω–µ –∑–Ω–∞—á–µ–Ω–Ω—è –∫–æ–¥–æ–≤–æ—ó —Ç–æ—á–∫–∏, –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—é—á–∏ –Ω–∞—Å—Ç—É–ø–Ω—ñ escape-–ø–æ—Å–ª—ñ–¥–æ–≤–Ω–æ—Å—Ç—ñ:
```python
a = '\xf1'          # a = '√±'
b = '\u2200'        # b = '‚àÄ'
c = '\U0001D122'    # c = 'ùÑ¢'
d = '\N{FOR ALL}'   # d = '‚àÄ'
```
[–ë–∞–∑–∞ –¥–∞–Ω–∏—Ö —Å–∏–º–≤–æ–ª—ñ–≤ Unicode](https://unicode.org/charts) —î –ø–æ—Å–∏–ª–∞–Ω–Ω—è–º –Ω–∞ –≤—Å—ñ –¥–æ—Å—Ç—É–ø–Ω—ñ –∫–æ–¥–∏ —Å–∏–º–≤–æ–ª—ñ–≤.

### –Ü–Ω–¥–µ–∫—Å—É–≤–∞–Ω–Ω—è —Ä—è–¥–∫—ñ–≤
–†—è–¥–∫–∏ –ø—Ä–∞—Ü—é—é—Ç—å —è–∫ –º–∞—Å–∏–≤ –¥–ª—è –¥–æ—Å—Ç—É–ø—É –¥–æ –æ–∫—Ä–µ–º–∏—Ö —Å–∏–º–≤–æ–ª—ñ–≤. –í–∏ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—î—Ç–µ —Ü—ñ–ª–æ—á–∏—Å–µ–ª—å–Ω–∏–π —ñ–Ω–¥–µ–∫—Å, –ø–æ—á–∏–Ω–∞—é—á–∏ –∑ 0. –í—ñ–¥‚Äô—î–º–Ω—ñ —ñ–Ω–¥–µ–∫—Å–∏ –≤–∫–∞–∑—É—é—Ç—å –ø–æ–∑–∏—Ü—ñ—é –≤—ñ–¥–Ω–æ—Å–Ω–æ –∫—ñ–Ω—Ü—è —Ä—è–¥–∫–∞. –ú–∏ –ø–æ–≤–µ—Ä–Ω–µ–º–æ—Å—è –¥–æ —Ü—å–æ–≥–æ —â–µ —Ä–∞–∑ –≤ –Ω–∞—Å—Ç—É–ø–Ω–æ–º—É —Ä–æ–∑–¥—ñ–ª—ñ –ø—Ä–æ —Å–ø–∏—Å–∫–∏.
```python
a = 'Hello world'
b = a[0]          # 'H'
c = a[4]          # 'o'
d = a[-1]         # 'd' (–∫—ñ–Ω–µ—Ü—å —Ä—è–¥–∫–∞)
```
–í–∏ —Ç–∞–∫–æ–∂ –º–æ–∂–µ—Ç–µ —Ä–æ–∑–¥—ñ–ª–∏—Ç–∏ –∞–±–æ –≤–∏–±—Ä–∞—Ç–∏ –ø—ñ–¥—Ä—è–¥–∫–∏, –≤–∫–∞–∑—É—é—á–∏ –¥—ñ–∞–ø–∞–∑–æ–Ω —ñ–Ω–¥–µ–∫—Å—ñ–≤ –∑–∞ –¥–æ–ø–æ–º–æ–≥–æ—é `:`.
```python
d = a[:5]     # 'Hello'
e = a[6:]     # 'world'
f = a[3:8]    # 'lo wo'
g = a[-5:]    # 'world'
```
–í—ñ–¥—Å—É—Ç–Ω—ñ —ñ–Ω–¥–µ–∫—Å–∏ –ø—Ä–∏–ø—É—Å–∫–∞—é—Ç—å –ø–æ—á–∞—Ç–æ–∫ –∞–±–æ –∫—ñ–Ω–µ—Ü—å —Ä—è–¥–∫–∞.

### –û–ø–µ—Ä–∞—Ü—ñ—ó –∑ —Ä—è–¥–∫–∞–º–∏
–ö–æ–Ω–∫–∞—Ç–µ–Ω–∞—Ü—ñ—è, –¥–æ–≤–∂–∏–Ω–∞, —á–ª–µ–Ω—Å—Ç–≤–æ —Ç–∞ —Ä–µ–ø–ª—ñ–∫–∞—Ü—ñ—è.
```python
# –ö–æ–Ω–∫–∞—Ç–µ–Ω–∞—Ü—ñ—è (+)
a = 'Hello' + 'World'   # 'HelloWorld'
b = 'Say ' + a          # 'Say HelloWorld'

# –î–æ–≤–∂–∏–Ω–∞ (len)
s = 'Hello'
len(s)                  # 5

# –ù–∞—è–≤–Ω—ñ—Å—Ç—å/–≤—ñ–¥—Å—É—Ç–Ω—ñ—Å—Ç—å (`in`, `not in`)
t = 'e' in s            # True
f = 'x' in s            # False
g = 'hi' not in s       # True

# –†–µ–ø–ª—ñ–∫–∞—Ü—ñ—è (s * n)
rep = s * 5             # 'HelloHelloHelloHelloHello'
```

### –ú–µ—Ç–æ–¥–∏ —Ä—è–¥–∫—ñ–≤
–†—è–¥–∫–∏ –º–∞—é—Ç—å –º–µ—Ç–æ–¥–∏, —è–∫—ñ –≤–∏–∫–æ–Ω—É—é—Ç—å —Ä—ñ–∑–Ω—ñ –æ–ø–µ—Ä–∞—Ü—ñ—ó –∑ —Ä—è–¥–∫–æ–≤–∏–º–∏ –¥–∞–Ω–∏–º–∏.

–ü—Ä–∏–∫–ª–∞–¥: –≤–∏–¥–∞–ª–µ–Ω–Ω—è –±—É–¥—å-—è–∫–æ–≥–æ –ø–æ—á–∞—Ç–∫–æ–≤–æ–≥–æ / –∫—ñ–Ω—Ü–µ–≤–æ–≥–æ –ø—Ä–æ–±—ñ–ª—É.
```python
s = '  Hello '
t = s.strip()     # 'Hello'
```

–ü—Ä–∏–∫–ª–∞–¥: –ø–µ—Ä–µ—Ç–≤–æ—Ä–µ–Ω–Ω—è —Ä–µ–≥—ñ—Å—Ç—Ä—É.
```python
s = 'Hello'
l = s.lower()     # 'hello'
u = s.upper()     # 'HELLO'
```

–ü—Ä–∏–∫–ª–∞–¥: –∑–∞–º—ñ–Ω–∞ —Ç–µ–∫—Å—Ç—É.
```python
s = 'Hello world'
t = s.replace('Hello' , 'Hallo')   # 'Hallo world'
```

**–ë—ñ–ª—å—à–µ —Ä—è–¥–∫–æ–≤–∏—Ö –º–µ—Ç–æ–¥—ñ–≤:**

–†—è–¥–∫–∏ –º–∞—é—Ç—å —à–∏—Ä–æ–∫–∏–π —Å–ø–µ–∫—Ç—Ä —ñ–Ω—à–∏—Ö –º–µ—Ç–æ–¥—ñ–≤ –¥–ª—è —Ç–µ—Å—Ç—É–≤–∞–Ω–Ω—è —Ç–∞ –º–∞–Ω—ñ–ø—É–ª—é–≤–∞–Ω–Ω—è —Ç–µ–∫—Å—Ç–æ–≤–∏–º–∏ –¥–∞–Ω–∏–º–∏. –ó—Ä–∞–∑–æ–∫ —Ü–∏—Ö –º–µ—Ç–æ–¥—ñ–≤:
```python
s.endswith(suffix)     # –ü–µ—Ä–µ–≤—ñ—Ä–∏—Ç–∏, —á–∏ –∑–∞–∫—ñ–Ω—á—É—î—Ç—å—Å—è —Ä—è–¥–æ–∫ —Å—É—Ñ—ñ–∫—Å–æ–º
s.find(t)              # –ü–µ—Ä—à–µ –ø–æ—è–≤–∞ t —É s
s.index(t)             # –ü–µ—Ä—à–µ –ø–æ—è–≤–∞ t —É s
s.isalpha()            # –ü–µ—Ä–µ–≤—ñ—Ä–∏—Ç–∏, —á–∏ —Å–∏–º–≤–æ–ª–∏ —î –∞–ª—Ñ–∞–≤—ñ—Ç–Ω–∏–º–∏
s.isdigit()            # –ü–µ—Ä–µ–≤—ñ—Ä–∏—Ç–∏, —á–∏ —Å–∏–º–≤–æ–ª–∏ —î —á–∏—Å–ª–æ–≤–∏–º–∏
s.islower()            # –ü–µ—Ä–µ–≤—ñ—Ä–∏—Ç–∏, —á–∏ —Å–∏–º–≤–æ–ª–∏ –Ω–∏–∂–Ω—å–æ–≥–æ —Ä–µ–≥—ñ—Å—Ç—Ä—É (–º–∞–ª–µ–Ω—å–∫—ñ –±—É–∫–≤–∏)
s.isupper()            # –ü–µ—Ä–µ–≤—ñ—Ä–∏—Ç–∏, —á–∏ —Å–∏–º–≤–æ–ª–∏ –≤–µ—Ä—Ö–Ω—å–æ–≥–æ —Ä–µ–≥—ñ—Å—Ç—Ä—É (–≤–µ–ª–∏–∫—ñ –±—É–∫–≤–∏)
s.join(slist)          # –ó'—î–¥–Ω–∞—Ç–∏ –≤—Å—ñ –µ–ª–µ–º–µ–Ω—Ç–∏ —Å–ø–∏—Å–∫—É –≤ —Ä—è–¥–æ–∫, –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—é—á–∏ s —è–∫ —Ä–æ–∑–¥—ñ–ª—å–Ω–∏–∫
s.lower()              # –ü–µ—Ä–µ—Ç–≤–æ—Ä–∏—Ç–∏ –≤ –Ω–∏–∂–Ω—ñ–π —Ä–µ–≥—ñ—Å—Ç—Ä
s.replace(old,new)     # –ó–∞–º—ñ–Ω–∏—Ç–∏ —Ç–µ–∫—Å—Ç
s.rfind(t)             # –®—É–∫–∞—Ç–∏ t –∑ –∫—ñ–Ω—Ü—è —Ä—è–¥–∫–∞
s.rindex(t)            # –®—É–∫–∞—Ç–∏ t –∑ –∫—ñ–Ω—Ü—è —Ä—è–¥–∫–∞
s.split([delim])       # –†–æ–∑–¥—ñ–ª–∏—Ç–∏ —Ä—è–¥–æ–∫ –Ω–∞ —Å–ø–∏—Å–æ–∫ –ø—ñ–¥—Ä—è–¥–∫—ñ–≤
s.startswith(prefix)   # –ü–µ—Ä–µ–≤—ñ—Ä–∏—Ç–∏, —á–∏ –ø–æ—á–∏–Ω–∞—î—Ç—å—Å—è —Ä—è–¥–æ–∫ –∑ –ø—Ä–µ—Ñ—ñ–∫—Å–∞
s.strip()              # –í–∏–¥–∞–ª–∏—Ç–∏ –ø—Ä–æ–±—ñ–ª–∏ —Å–ø–µ—Ä–µ–¥—É/–∑–∑–∞–¥—É
s.upper()              # –ü–µ—Ä–µ—Ç–≤–æ—Ä–∏—Ç–∏ —É –≤–µ—Ä—Ö–Ω—ñ–π —Ä–µ–≥—ñ—Å—Ç—Ä
```

### –ó–º—ñ–Ω–Ω—ñ—Å—Ç—å —Ä—è–¥–∫–∞
–†—è–¥–∫–∏ —î "–Ω–µ–∑–º—ñ–Ω–Ω–∏–º–∏" –∞–±–æ –¥–æ—Å—Ç—É–ø–Ω–∏–º–∏ –ª–∏—à–µ –¥–ª—è —á–∏—Ç–∞–Ω–Ω—è. –ü—ñ—Å–ª—è —Å—Ç–≤–æ—Ä–µ–Ω–Ω—è –∑–Ω–∞—á–µ–Ω–Ω—è –Ω–µ –º–æ–∂–Ω–∞ –∑–º—ñ–Ω–∏—Ç–∏.
```python
>>> s = 'Hello World'
>>> s[1] = 'a'
Traceback (most recent call last):
File "<stdin>", line 1, in <module>
TypeError: 'str' object does not support item assignment
>>>
```
**–£—Å—ñ –æ–ø–µ—Ä–∞—Ü—ñ—ó —Ç–∞ –º–µ—Ç–æ–¥–∏, —è–∫—ñ –º–∞–Ω—ñ–ø—É–ª—é—é—Ç—å —Ä—è–¥–∫–æ–≤–∏–º–∏ –¥–∞–Ω–∏–º–∏, –∑–∞–≤–∂–¥–∏ —Å—Ç–≤–æ—Ä—é—é—Ç—å –Ω–æ–≤—ñ —Ä—è–¥–∫–∏.**

### –ü–µ—Ä–µ—Ç–≤–æ—Ä–µ–Ω–Ω—è —Ä—è–¥–∫—ñ–≤
–í–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–π—Ç–µ `str()`, —â–æ–± –ø–µ—Ä–µ—Ç–≤–æ—Ä–∏—Ç–∏ –±—É–¥—å-—è–∫–µ –∑–Ω–∞—á–µ–Ω–Ω—è –≤ —Ä—è–¥–æ–∫. –†–µ–∑—É–ª—å—Ç–∞—Ç–æ–º —î —Ä—è–¥–æ–∫, —â–æ –º—ñ—Å—Ç–∏—Ç—å —Ç–æ–π —Å–∞–º–∏–π —Ç–µ–∫—Å—Ç, —è–∫–∏–π –±—É–≤ –±–∏ —Å—Ç–≤–æ—Ä–µ–Ω–∏–π –æ–ø–µ—Ä–∞—Ç–æ—Ä–æ–º `print()`.
```python
>>> x = 42
>>> str(x)
'42'
>>>
```

### –ë–∞–π—Ç–æ–≤—ñ —Ä—è–¥–∫–∏
–†—è–¥–æ–∫ —ñ–∑ 8-–±—ñ—Ç–æ–≤–∏—Ö –±–∞–π—Ç—ñ–≤, —è–∫–∏–π –∑–∞–∑–≤–∏—á–∞–π –∑—É—Å—Ç—Ä—ñ—á–∞—î—Ç—å—Å—è –ø—Ä–∏ –Ω–∏–∑—å–∫–æ—Ä—ñ–≤–Ω–µ–≤–æ–º—É –≤–≤–æ–¥—ñ/–≤–∏–≤–æ–¥—ñ, –∑–∞–ø–∏—Å—É—î—Ç—å—Å—è —Ç–∞–∫–∏–º —á–∏–Ω–æ–º:
```python
data = b'Hello World\r\n'
```
–ü–æ—Å—Ç–∞–≤–∏–≤—à–∏ –º–∞–ª–µ–Ω—å–∫—É b –ø–µ—Ä–µ–¥ –ø–µ—Ä—à–æ—é –ª–∞–ø–∫–æ—é, –≤–∏ –≤–∫–∞–∑—É—î—Ç–µ, —â–æ —Ü–µ —Ä—è–¥–æ–∫ –±–∞–π—Ç—ñ–≤, –∞ –Ω–µ —Ç–µ–∫—Å—Ç–æ–≤–∏–π —Ä—è–¥–æ–∫.
–ë—ñ–ª—å—à—ñ—Å—Ç—å –∑–≤–∏—á–∞–π–Ω–∏—Ö —Ä—è–¥–∫–æ–≤–∏—Ö –æ–ø–µ—Ä–∞—Ü—ñ–π –ø—Ä–∞—Ü—é—é—Ç—å.
```python
len(data)                         # 13
data[0:5]                         # b'Hello'
data.replace(b'Hello', b'Cruel')  # b'Cruel World\r\n'
```
–Ü–Ω–¥–µ–∫—Å—É–≤–∞–Ω–Ω—è –¥–µ—â–æ –≤—ñ–¥—Ä—ñ–∑–Ω—è—î—Ç—å—Å—è, –æ—Å–∫—ñ–ª—å–∫–∏ –≤–æ–Ω–æ –ø–æ–≤–µ—Ä—Ç–∞—î –∑–Ω–∞—á–µ–Ω–Ω—è –±–∞–π—Ç—ñ–≤ —É –≤–∏–≥–ª—è–¥—ñ —Ü—ñ–ª–∏—Ö —á–∏—Å–µ–ª.
```python
data[0]   # 72 (ASCII –∫–æ–¥ –≤–µ–ª–∏–∫–æ—ó –±—É–∫–≤–∏ 'H')
```
–ü–µ—Ä–µ—Ç–≤–æ—Ä–µ–Ω–Ω—è –≤/–∑ —Ç–µ–∫—Å—Ç–æ–≤–∏—Ö —Ä—è–¥–∫—ñ–≤.
```python
text = data.decode('utf-8') # –±–∞–π—Ç–∏ -> —Ç–µ–∫—Å—Ç
data = text.encode('utf-8') # —Ç–µ–∫—Å—Ç -> –±–∞–π—Ç–∏
```
